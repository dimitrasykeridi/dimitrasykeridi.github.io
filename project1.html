Project 1

Analysis of Greek Airbnb Listings using SQL

This project focuses on analyzing Airbnb Listings, Reviews, and Bookings data using SQL.

The Opportunity:

The objective was to extract data from Airbnbâ€™s official website, build a Data Warehouse, perform data cleaning and transformation and derive insights through SQL queries.

Step 1 : Data Extraction and Loading

The data are from the official site of Airbnb and they are licensed under a Creative Commons Attribution 4.0 International License. The extracted data was loaded into a database and a data warehouse was built from scratch using SQL Server Management Studio

Step 2 : Data Cleaning

During the Data Warehouse creation process, I replaced specific values with numeric equivalents, substituted null values with zeros where applicable, and used various SQL functions to modify data types.

Step 3: Data Modelling

To set up the schema for this analysis, I developed Fact and Dimension Tables to structure the data, resulting in a total of seven tables.

Key Insights:

Pricing & Popular Neighborhoods

  • Overall Trends : Across 7,335 neighborhoods in Greece, the average Airbnb listing price is $121.45.

  • Acropolis Insights: The Acropolis neighborhood has the highest average listing price at $415, attributed to its historical significance.

Larger Group Accommodation

  • Acropolis properties for up to 8 guests attract larger groups or families

Superhost Profile Behavior

  • Superhosts are 3.7% more likely to have profile pictures (98% vs. 94%), which may slightly boost their success.

Host Ownership & Property Type

  • Non-superhosts own more properties, but both groups prefer listing entire apartments with two beds.

Top Hosts by Reviews & Booking

  • Hosts like George and Toni have the highest average number of bookings and reviews, showing consistent guest engagement.

Stay Duration Insights

  • Hotels average 1,037 nights and entire apartments average 912 nights for maximum bookings.

Seasonal Trends

  • The number of bookings spikes in September and October.

The Outcome:

This project provided valuable insights into Airbnb listings in Greece by analyzing pricing trends, accommodation preferences, and host behaviors. Key findings include the high average listing price in Acropolis, the appeal of large accommodations for families, and the slight advantage of profile pictures for superhosts. The analysis also highlighted seasonal booking trends and preferences for extended stays, offering actionable recommendations for hosts and potential investors.
